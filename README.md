# RedFace

> **RedFace makes operations verifiable.**

RedFace is an **Operational Infrastructure Platform**. Trust is not assumed — it is earned through verifiable operations.

**Shield** is the first reference implementation. **Commerce** is the second. The **Operational Kernel** is the company.

---

## Architecture v1.0 — Ratified

Architecture is internally consistent and **frozen**. Changes require constitutional amendment.

Primary focus:

- **Prove operations:** [First Operational Demonstration](docs/operations/FIRST-OPERATIONAL-DEMONSTRATION.md)
- **Win the market:** [Adoption Strategy](docs/business/ADOPTION-STRATEGY.md)

```bash
npm install
npm run db:up
npm run db:migrate
npm run mvp:hijacking      # Demonstration A: Vehicle Recovery
npm run playback:demo        # Mission playback timeline
npm run sim:recovery         # Mission simulator (RSP SDK)
npm run kernel:dev             # API :3000
```

See [IMPLEMENTATION.md](IMPLEMENTATION.md) for CTO roadmap and Q1 objectives.
