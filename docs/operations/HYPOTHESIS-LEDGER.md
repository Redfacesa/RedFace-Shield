## Status values

| Status | Meaning |
|--------|---------|
| Proposed | Not yet tested |
| Testing | Active in a deployment |
| Accepted | Evidence supports hypothesis |
| Rejected | Evidence contradicts — revert or defer |
| Unknown | Collecting — need more missions |
